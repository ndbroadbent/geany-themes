Geany Themes

A collection of color schemes for Geany.

UPDATE: the geany-themes branches were renamed to follow the correct Geany version number. Use branch `0.20` for Geany 0.20, branch `0.21` for Geany 0.21, and `master` if you're building Geany from its `master` Git branch.

UPDATE: after Geany 0.21, the filedefs from geany-themes have been merged into Geany proper so only the colour scheme *.conf files are needed.

UPDATE: if you aren't using the development version of Geany from Git or Subversion, you need to use the deprecated branch instead of the master branch. The current master branch is using a new feature (see below) since the last Geany release and so is not compatible.

UPDATE: Since Geany r5596, the substitution and mappings hackery is no longer needed. Geany now supports copying the styling of one filetype to another. I've removed those parts from geany-themes, simplifying the entire project. Sweet!

geany-themes is a set of color schemes containing some really nice themes. I have ported almost all of the existing Geany themes I could find.

Some of the new themes were ported from the gedit-themes and I plan to eventually port the rest of them. There is also a bunch of available themes for vim and TextMate that would also be nice to port.

You can create your own theme by using one of the existing themes as a starting point and there is more information in the Geany Manual. No programming skills are required at all to create your own theme.

Dependencies

Geany 0.20 or higher.

Note: make sure you checkout the correct branch of geany-themes for the Geany version you are running. See the README.md file for more information.
